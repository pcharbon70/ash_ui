## Submitting an RFC

To submit a new RFC:

1. Read [getting-started.md](getting-started.md)
2. Copy [templates/rfc-template.md](templates/rfc-template.md)
3. Create your RFC as `rfcs/RFC-XXXX-title.md`
4. Submit a PR with your RFC
5. Participate in the discussion
